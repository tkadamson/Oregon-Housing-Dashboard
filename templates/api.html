Oregon Housing Dashboard API

Welcome to the API that runs the Housing Dashboard. The data here was taken from the American Community Survey (2015-2019). For each Oregon county, the API contains population, median income, median rent, number of renters, number of homeowners, new housing units built since 2013 and county fips. The US and Oregon-wide pages include only median rents for years 2015 to 2019.

To use the api, use 'api/v1/us' for US data, 'api/v1/or' for Oregon, and 'api/v1/county' where county is your desired county in Oregon.
